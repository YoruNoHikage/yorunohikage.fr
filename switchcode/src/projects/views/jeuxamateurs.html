JeuxAmateurs

À propos du site

Introduction

JeuxAmateurs est un site d'actualité, de tests et de vidéos sur les jeux vidéo amateurs. Actuellement le site n'est plus disponible mais on espère revenir bientôt.

Technologies & Tools

JeuxAmateurs a beaucoup évolué technologiquement, au fur et à mesure de l’apprentissage en autodidacte de l’équipe. D’abord en pur xHTML / CSS (avec le doctype à rallonge de l’époque), PHP est venu par la suite rendre le site dynamique. Jusqu’à la dernière version en ligne, le site n’utilisait aucun framework (comprendre outil préfabriqué aidant le développement) et ne suivait même pas le modèle d’architecture bien connu MVC (là-dessus, on vous laisse checker wikipédia).
Une refonte du site est prévue, utilisant le framework Symfony afin de construire une API REST côté serveur ouverte à tous et Backbone.js / Marionette.js pour le côté client.

Histoire

L’histoire JeuxAmateurs.fr commença avec la réalisation d’un site de présentation des projets de SwitchCode. En y appliquant les premières compétences en informatique qu’ils avaient pu acquérir, les deux compères de l’équipe firent un site plutôt simple contenant quelques modules basiques, les news et le sondage. Puis le site évolua avec l’idée de représenter tous les jeux amateurs trouvés sur le net afin de faire une base commune de ce qui existait déjà éparpillés ailleurs. Ainsi la V2 sortie avec un design différent et un premier système d’authentification. Les tests, aperçus et commentaires rejoignirent les modules présents. Plus tard, avec un design repensé mais gardant les couleurs propres à l’image du site, on vit le lancement des vidéos et des AJVA (Actualités des Jeux Vidéo Amateurs) et l’achat du nom de domaine jeuxamateurs.fr le 11 novembre 2010.

Par la suite, le 4 septembre 2014, c’est jeuxamateurs.com qui est acheté pour y placer un site temporaire contenant quelques articles sur les jeux vidéo amateurs le temps de relancer le site.
